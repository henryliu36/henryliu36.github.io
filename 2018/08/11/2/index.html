Posted on 2018-08-11

诚然可以使用文件夹右键Bash，在该文件夹-init一个新的repo，但是使用可视化的Git Desktop也相当便捷。

Git客户端方法

  1. Git官网可以直接下载适配你的电脑版本的Git
  2. 下载之后，建立一个Repo，File-New Repository
  3. 在该文件夹内部建立你所要上传的文件夹，之后File-add new repository
  4. 在右下角的Summary随便写点description并commit到master（当前建立的目录）
  5. 右端有Publish的指令

    切记要选public而不是private，不然会产生Exit code 1错误

  6. 在Github则可以看到你所建立的repo内容
