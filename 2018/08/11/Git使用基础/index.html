Git获取别人的项目并进行修改

  • 自己使用
    进入别人的仓库，使用右上角的Fork，便可以将别人的仓库copy到自己的github的仓库中。随后可以在github用edit进行修改，并commit change。

  • 协作修改

  1. 如果想修改原仓库文件，需要在自己所copy的仓库中，做出修改之后，点击pull request-new pull request， 然后出现comparing changes看所做的修改。
  2. 在所有者的仓库中，可以看到pull request列表出现新文件，此时所有者（其他合作者也能看到）应进行审核。如无问题，则可在pull request列表下方Merge pull request-comfirm merge
